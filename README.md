## 📚 Projects

### 1. 🤖 [AI News Telegram Agent](./01-ai-news-telegram-agent)

**Daily AI news digest delivered to Telegram automatically at 9 AM.**

- 📰 Aggregates 2000+ articles from 6 AI sources daily
- 🧠 Uses Groq (Llama 3.3 70B) for intelligent summarization
- 📩 Auto-delivers formatted digest via Telegram bot
- ☁️ Runs 24/7 on free Render cloud
- 💰 **Cost: $0/month**
